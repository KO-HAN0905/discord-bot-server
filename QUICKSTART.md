# 🚀 빠른 시작 가이드 (초보자용)

> 디스코드 봇을 처음 사용하시나요? 이 가이드를 따라하시면 5분 안에 봇을 사용할 수 있습니다!

---

## 📌 준비물

✅ 디스코드 계정  
✅ 봇이 설치된 서버의 관리자 권한  
✅ 봇이 이미 실행 중 (터미널에 "봇 준비됨" 메시지 확인)

---

## 🎯 1단계: 개발자 모드 켜기 (1분)

채널 ID를 복사하려면 개발자 모드를 켜야 합니다.

1. 디스코드 앱 열기
2. ⚙️ **설정** (왼쪽 아래 톱니바퀴) 클릭
3. **앱 설정** → **고급** 메뉴로 이동
4. **개발자 모드** 스위치를 켜기 (파란색으로 변경)
5. 설정 창 닫기

✅ 완료! 이제 모든 채널/사용자를 우클릭하면 "ID 복사" 메뉴가 보입니다.

---

## 🎯 2단계: 봇과 인사하기 (30초)

디스코드 서버의 아무 채널에서 다음 명령어를 입력하세요:

```
!메뉴
```

**메인 메뉴가 나타납니다!** 이제 버튼만 클릭하면 모든 기능을 사용할 수 있습니다! 🎉

다른 유용한 명령어:

```
!도움말
```

봇이 모든 명령어 목록을 보여줍니다!

```
!정보
```

봇의 정보를 확인할 수 있습니다.

---

## 🎯 3단계: 채널 설정하기 (2분)

봇이 뉴스와 D-Day를 발송할 채널을 설정합니다.

### 방법 1: UI 버튼 사용 (추천) 👍

```
!설정ui
```

1. 봇이 버튼이 있는 메시지를 보냅니다
2. **📰 뉴스 채널 설정** 버튼 클릭
3. 뉴스를 받고 싶은 채널을 우클릭 → **ID 복사**
4. 팝업 창에 ID 붙여넣기 → 제출
5. 같은 방법으로 **📅 D-Day 채널 설정**

✅ 완료! 이제 봇이 설정한 채널에 자동으로 소식을 전합니다.

### 방법 2: 명령어 사용

```
!채널설정 뉴스 #일반채팅
!채널설정 디데이 #공지채널
```

채널을 멘션(@)하면 자동으로 설정됩니다.

---

## 🎯 4단계: 주요 기능 사용하기 (2분)

### 🔔 알람 추가

```
!알람ui
```

버튼을 클릭하여 쉽게 알람을 추가할 수 있습니다!

**예시:**
- "➕ 알람 추가" 버튼 클릭
- 이름: `회의`
- 시간: `14:30`
- 반복: `daily` (매일) 또는 `once` (1회)

### 📋 과제 추가

```
!과제ui
```

일일 과제와 주간 과제를 관리할 수 있습니다.

**예시:**
- "➕ 과제 추가" 버튼 클릭
- 종류: `daily` 또는 `weekly`
- 이름: `영어공부`

### 📅 D-Day 추가

```
!디데이 ui
```

중요한 날짜를 카운트다운할 수 있습니다.

**예시:**
- "➕ D-Day 추가" 버튼 클릭
- 이름: `시험`
- 날짜: `2026-03-15` (YYYY-MM-DD 형식)
- 메시지: `중간고사` (선택사항)

### 📰 게임 뉴스

```
!뉴스ui
```

최신 게임 뉴스를 확인하고 공지할 수 있습니다.

## 💡 팁과 요령

### 가장 쉬운 사용법

```
!메뉴
```

이 명령어 하나면 모든 기능을 버튼으로 사용할 수 있습니다!

### UI 버튼 vs 명령어

- **UI 버튼** (추천): 초보자도 쉽게 사용 가능, 실수 적음
- **명령어**: 빠른 입력, 고급 사용자용

### 모든 UI 명령어

**가장 쉬운 방법:**
```
!메뉴          # 메인 메뉴 (모든 기능 버튼)
```

**개별 UI:**
```
!알람ui        # 알람 관리
!과제ui        # 과제 관리
!디데이 ui     # D-Day 관리
!뉴스ui        # 게임 뉴스 관리
!설정ui        # 채널 설정
```

### 자주 사용하는 명령어

```
!도움말        # 전체 명령어 확인
!정보          # 봇 정보
```

## 🆘 문제 해결

### Q1: 봇이 응답하지 않아요

**해결방법:**
1. 봇이 온라인 상태인지 확인 (우측 멤버 목록에서 초록불)
2. 터미널에 "봇 준비됨" 메시지가 있는지 확인
3. 명령어 앞에 `!`를 붙였는지 확인

### Q2: "권한이 부족합니다" 오류

**해결방법:**
- 채널 설정, 뉴스 공지 등은 **관리자 권한**이 필요합니다
- 서버 설정 → 역할에서 관리자 권한 확인

### Q3: 채널 ID를 복사할 수 없어요

**해결방법:**
1. 개발자 모드가 켜져 있는지 확인 (1단계 참조)
2. 채널을 우클릭했는지 확인 (모바일에서는 길게 누르기)

### Q4: 버튼이 작동하지 않아요

**해결방법:**
- 봇을 최근에 재시작했는지 확인
- UI 명령어를 다시 입력해보세요 (예: `!알람ui`)

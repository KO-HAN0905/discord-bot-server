# 🤖 Discord Multi-Function Bot

완벽한 디스코드 서버 관리 봇입니다.

## ✨ 주요 기능

### 1. 🔔 알람 관리
- 시간별 알람 설정
- 매일 반복 또는 일회성 알람
- 알람 추가/삭제/목록 조회
- 스케줄 자동 관리

### 2. 📰 게임 뉴스
- 최신 게임 정보 자동 업데이트
- 공지 형식 임베드 출력
- 6시간마다 자동 갱신

### 3. 📋 과제 관리
- 일일 과제 & 주간 과제 구분
- 0~100% 진행도 게이지
- 진행 상황 실시간 추적
- 과제 완료 기능

### 4. 📅 D-Day 관리
- 목표 날짜 설정 및 관리
- 엑셀 파일로 자동 저장
- 특정 채널에 공지 발송
- 남은 날짜 자동 계산

### 5. 🔄 데이터 동기화
- JSON 파일 자동 저장
- 엑셀 파일 동기화
- 구조화된 데이터 관리

### 6. ⚙️ 채널 설정 (관리자)
- 뉴스 채널 동적 설정
- D-Day 채널 동적 설정
- 설정 관리 및 초기화

## 📦 설치

### 요구사항
- Python 3.9+
- pip (Python 패키지 매니저)

### 빠른 시작

```bash
# 1. 프로젝트 폴더 이동
cd f:\A

# 2. 의존성 설치
pip install -r requirements.txt

# 3. .env 파일 설정
# .env 파일을 열고 DISCORD_BOT_TOKEN 설정

# 4. 봇 실행
python bot.py
```

## 🚀 사용법

### ⭐ 가장 쉬운 방법 - 메인 메뉴 사용

```bash
!메뉴          # 메인 메뉴 버튼 표시 (모든 기능 접근)
!시작          # !메뉴와 동일
```

메인 메뉴에서 버튼 하나만 클릭하면 모든 기능을 사용할 수 있습니다!

### 기본 명령어

```bash
!도움말          # 전체 명령어 확인
!정보            # 봇 정보 확인
!대시보드        # 모든 기능 한눈에 보기 (초보자 추천)
```

### 알람 관리

**🎯 UI 버튼 방식 (권장):**
```bash
!알람ui          # 버튼식 UI 표시 (추가, 목록, 삭제 버튼)
```
- 모달을 통해 알람 이름, 시간, 반복 설정 입력
- 버튼 클릭으로 간편한 관리

**명령어 방식:**
```bash
!알람 추가 회의 14:30 daily    # 매일 14:30 알람
!알람 목록                      # 알람 목록 확인
!알람 삭제 회의                # 알람 삭제
!알람 새로고침                 # 스케줄 업데이트
```

### 게임 뉴스

**🎯 UI 버튼 방식 (권장):**
```bash
!뉴스ui          # 버튼식 UI 표시 (목록, 새로고침, 공지 버튼)
```
- 버튼 클릭으로 뉴스 조회 및 관리

**명령어 방식:**
```bash
!게임뉴스        # 최신 뉴스 확인
!뉴스공지        # 공지로 발송
```

### 과제 관리

**🎯 UI 버튼 방식 (권장):**
```bash
!과제ui          # 버튼식 UI 표시 (추가, 목록, 삭제/완료 버튼)
```
- 모달을 통해 과제 종류(일일/주간)와 이름 입력
- 버튼 클릭으로 간편한 과제 조회

**명령어 방식:**
```bash
!과제 일일추가 영어공부                # 일일 과제 추가
!과제 진행도 일일 50 영어공부         # 진행도 50% 설정
!과제 완료 일일 영어공부              # 과제 완료
!과제 목록                             # 과제 목록 확인
```

### D-Day 관리

**🎯 UI 버튼 방식 (권장):**
```bash
!디데이 ui       # 버튼식 UI 표시 (추가, 목록, 새로고침, 삭제 버튼)
```
- 모달에서 직접 날짜와 메시지 입력
- 목록 보기, 새로고침도 버튼으로 간편하게 관리

**명령어 방식:**
```bash
!디데이 추가 시험 2026-03-15    # D-Day 추가
!디데이 목록                      # D-Day 목록
!디데이 공지                      # 공지로 발송
!디데이 새로고침                 # 업데이트
```

### 채널 설정 (관리자 전용)

**🎯 UI 버튼 방식 (권장):**
```bash
!설정ui          # 버튼식 UI 표시 (채널 설정, 확인, 초기화 버튼)
```
- 개발자 모드를 활성화하고 채널 ID를 복사하여 입력
- 간편한 버튼 클릭으로 모든 설정 관리
- 상세한 도움말 제공

**명령어 방식:**
```bash
!채널설정 뉴스 #채널             # 뉴스 채널 설정
!채널설정 디데이 #채널           # D-Day 채널 설정
!채널설정 확인                   # 현재 설정 확인
!채널설정 리셋                   # 설정 초기화
```

### 시스템 관리 (관리자 전용)

```bash
!동기화          # 봇 재부팅 및 모듈 재로드
```
- 코드 수정 후 봇을 다시 시작하지 않고 바로 적용
- 모든 모듈을 자동으로 재로드
- 관리자 권한 필요

## ⚙️ 설정

### .env 파일 설정

```env
DISCORD_BOT_TOKEN=여기에_봇_토큰
GAME_NEWS_CHANNEL_ID=뉴스채널ID
DDAY_CHANNEL_ID=디데이채널ID
```

### 봇 토큰 얻기

1. [Discord Developer Portal](https://discord.com/developers/applications) 방문
2. "New Application" 클릭
3. "Bot" 탭에서 "Add Bot" 클릭
4. "TOKEN" 섹션에서 토큰 복사

## 🛠️ 개발

### 새로운 기능 추가

`cogs/` 폴더에 새 파일 생성:

```python
import discord
from discord.ext import commands

class NewFeature(commands.Cog):
    def __init__(self, bot):
        self.bot = bot
    
    @commands.command(name="명령어")
    async def command(self, ctx):
        await ctx.send("응답")

async def setup(bot):
    await bot.add_cog(NewFeature(bot))
```

## 설치 및 실행

### 1단계: 필수 요구사항
- Python 3.9 이상
- 디스코드 계정 및 서버

### 2단계: 봇 설정
```bash
# 프로젝트 폴더에서
cd f:\A
```

### 3단계: 환경 변수 설정
`.env` 파일을 열고 다음을 입력:
```
DISCORD_BOT_TOKEN=여기에_디스코드_봇_토큰_입력
GAME_NEWS_CHANNEL_ID=뉴스채널ID
DDAY_CHANNEL_ID=디데이채널ID
```

### 4단계: 봇 실행
```bash
python bot.py
```

터미널에 "✅ BotName이 로그인했습니다!"라는 메시지가 나타나면 성공!

## 봇 초대

### 디스코드 개발자 포털에서 봇 토큰 얻기
1. https://discord.com/developers/applications 방문
2. "New Application" 클릭
3. 봇 이름 입력
4. "Bot" 탭에서 "Add Bot" 클릭
5. "TOKEN" 섹션의 토큰 복사
6. `.env` 파일에 붙여넣기

### 서버에 봇 초대
1. "OAuth2" → "URL Generator" 이동
2. Scopes: `bot` 선택
3. Permissions: 다음 항목 선택:
   - Send Messages
   - Embed Links
   - Manage Messages
   - Read Message History
4. 생성된 URL 복사 후 브라우저에서 열기

## 각 기능 상세 설명

### 1️⃣ 알람 관리

**🎯 UI 버튼 방식 (권장):**
```
!알람ui
```
1. "➕ 알람 추가" 버튼 클릭 → 모달에서 정보 입력
2. "📋 목록 보기" 버튼 클릭 → 등록된 알람 확인
3. "🔄 새로고침" 버튼 클릭 → 스케줄 업데이트
4. "❌ 삭제" 버튼 클릭 → 삭제 방법 안내

**명령어 방식:**

**알람 추가:**
```
!알람 추가 회의 14:30 daily
```
- `회의`: 알람 이름
- `14:30`: 시간 (24시간 형식)
- `daily` 또는 `once`: 매일 반복 또는 1회만

**알람 목록:**
```
!알람 목록
```

**알람 삭제:**
```
!알람 삭제 회의
```

**알람 새로고침:**
```
!알람 새로고침
```

---

### 5️⃣ 채널 설정 (관리자 전용)

**🎯 UI 버튼 방식 (권장):**
```
!설정ui
```

**사용 방법:**
1. **개발자 모드 활성화**
   - 디스코드 설정 → 앱 설정 → 고급 → **개발자 모드** ON

2. **채널 ID 복사**
   - 설정할 채널 우클릭 → **ID 복사**

3. **버튼으로 설정**
   - "📰 뉴스 채널 설정" 또는 "📅 D-Day 채널 설정" 버튼 클릭
   - 복사한 채널 ID 입력

4. **현재 설정 확인**
   - "📋 현재 설정 확인" 버튼으로 언제든지 확인 가능

5. **도움말 보기**
   - "❓ 도움말" 버튼으로 자세한 가이드 확인

**명령어 방식:**
```
!채널설정 뉴스 #채널명
!채널설정 디데이 #채널명
!채널설정 확인
!채널설정 리셋
```

### 2️⃣ 게임 뉴스

**🎯 UI 버튼 방식 (권장):**
```
!뉴스ui
```
1. "📋 뉴스 목록" 버튼 클릭 → 최신 뉴스 확인
2. "🔄 뉴스 새로고침" 버튼 클릭 → 뉴스 업데이트
3. "📢 뉴스 공지" 버튼 클릭 → 채널에 공지 발송 (관리자 전용)

**명령어 방식:**

**뉴스 확인:**
```
!게임뉴스
```

**뉴스 공지:**
```
!뉴스공지
```
채널에 공지 형식으로 최신 뉴스를 발송합니다.

---

### 3️⃣ 과제 관리

**🎯 UI 버튼 방식 (권장):**
```
!과제ui
```
1. "➕ 과제 추가" 버튼 클릭 → 모달에서 과제 종류(일일/주간)와 이름 입력
2. "📋 목록 보기" 버튼 클릭 → 등록된 과제 확인
3. "❌ 삭제/완료" 버튼 클릭 → 삭제/완료 방법 안내

**명령어 방식:**

**일일 과제 추가:**
```
!과제 일일추가 영어공부
```

**주간 과제 추가:**
```
!과제 주간추가 프로젝트완성
```

**진행도 설정:**
```
!과제 진행도 일일 50 영어공부
```
진행도는 0~100% 사이의 값입니다.

**과제 완료:**
```
!과제 완료 일일 영어공부
```

**과제 목록:**
```
!과제 목록
!과제 목록 일일
!과제 목록 주간
```

진행도 바 표시:
- █ = 완료됨
- ░ = 남은 부분

### 4️⃣ D-Day 관리

**UI 버튼 (관리자 전용) - 추천:**
```
!디데이 ui
```
버튼식 인터페이스로 쉽게 관리할 수 있습니다.

#### UI 버튼 기능:
- **➕ D-Day 추가** - 팝업 모달에서 날짜와 메시지 입력
- **📋 목록 보기** - 현재 등록된 D-Day 목록 확인
- **🔄 새로고침** - D-Day 값 자동 업데이트
- **❌ 삭제** - 명령어로 삭제

**모달 입력 양식:**
- 이름: D-Day 이름 (필수)
- 목표 날짜: YYYY-MM-DD 형식 (필수)
- 메시지: 설명 또는 메모 (선택)

**명령어 방식:**

**D-Day 추가:**
```
!디데이 추가 시험 2026-03-15
```

**D-Day 목록:**
```
!디데이 목록
```

결과 예시:
- `⏳ D+30일` - 남은 날짜
- `🎉 D-Day!` - 오늘이 목표날짜
- `📍 -5일 (경과)` - 지난 날짜

**D-Day 공지:**
```
!디데이 공지 #공지
```

**D-Day 삭제:**
```
!디데이 삭제 시험
```

**D-Day 새로고침:**
```
!디데이 새로고침
```

## 자주 묻는 질문

### Q: 명령어가 반응하지 않습니다
**A:** 다음을 확인하세요:
1. 봇이 실행 중인가? (터미널 확인)
2. 명령어 앞에 `!`가 붙어있는가?
3. 봇에게 메시지 보내기 권한이 있는가?

### Q: 알람이 작동하지 않습니다
**A:** 
1. 봇이 실행 중이어야 합니다
2. `!알람 새로고침` 명령어를 실행하세요
3. 시간 형식을 확인하세요 (HH:MM, 24시간 형식)

### Q: D-Day 엑셀 파일을 직접 수정할 수 있나요?
**A:** 가능하지만, 수정 후 봇을 재시작하거나 `!디데이 새로고침`을 실행해야 합니다.

### Q: 여러 서버에서 봇을 사용할 수 있나요?
**A:** 네! 현재 구조에서는 모든 서버가 같은 데이터를 공유합니다. 
서버별로 분리하려면 .env 파일을 수정하거나 고급 설정이 필요합니다.

### Q: 봇에 새로운 기능을 추가하고 싶습니다
**A:** `cogs/` 폴더에 새로운 Python 파일을 추가하면 됩니다. 
자세한 방법은 discord.py 공식 문서를 참고하세요.

## 문제 해결

### 오류: "TOKEN is invalid"
```
해결: .env 파일의 토큰을 다시 확인하세요
```

### 오류: "Cannot find module 'discord'"
```
해결: pip install -r requirements.txt 실행
```

### 데이터가 저장되지 않음
```
해결: data/ 폴더의 권한을 확인하세요
```

# 🧮 대미지 계산기 - UI 가이드

## 📌 빠른 시작

### 기본 명령어
```
!계산기          # 임베드 UI 계산기 시작 (초보자 추천)
!빠른계산 "빌드명"   # 빠른 계산 (명령어 입력 필요)
!대미지 "빌드명"     # 기존 명령어 계산
```

## 🎯 UI 계산기 사용 방법

### 1️⃣ 시작
```
!계산기
```
- 임베드 메시지가 나타나고 빌드 선택 드롭다운이 표시됩니다.

### 2️⃣ 빌드 선택
빌드 목록에서 원하는 빌드를 클릭하세요:
- 🏹 M82A1 루퍼스 크리 - 원거리 스나이퍼, 높은 피해
- 🔫 데저트이글 백상아리 크리 - 빠른 공격, 높은 DPS
- ⚔️ 신화검 탱커 - 생존 중심, 안정적
- 🔮 마법사 원소 - 범위 공격, 화려한 연출
- 💀 극대율 풀극 - 최고 DPS, 높은 위험도

### 3️⃣ 적 레벨 선택
적의 레벨을 선택합니다:
- 일반 몬스터 (레벨 15)
- 강한 몬스터 (레벨 20)
- 일반 보스 (레벨 25) ⭐ 기본값
- 강한 보스 (레벨 30)
- 매우 강한 보스 (레벨 40)

### 4️⃣ 결과 확인
계산 결과가 임베드로 표시됩니다:
```
🟢 계산 결과 1번
━━━━━━━━━━━━━━━━━━━
📋 기본 정보
- 적 레벨: 25
- 적 방어력: 60
- 공격속도: 1.1회/초

💥 대미지
- 일반: 367
- 극대: 1,100
- 평균: 916

⚡ 극대율
- 확률: 75.0%
- 배율: 3.00x

🔥 최종 결과
- 최종 피해: 780
- DPS: 858 💥
```

### 5️⃣ 또 다른 계산 (비교하기)
"➕ 또 다른 계산" 버튼을 클릭하여 다른 빌드를 계산합니다.

### 6️⃣ 계산값 비교
두 번째 계산 후 "⚔️ 비교" 버튼으로 비교합니다:
```
⚔️ 대미지 계산값 비교
━━━━━━━━━━━━━━━━━━━
🟢 계산1: 극대율 풀극 DPS: 858
🔵 계산2: 데저트이글 백상아리 DPS: 555

🏆 승자
- DPS: 🟢 계산1
- 피해: 🟢 계산1

📊 상세 비교
항목            🟢 계산1      🔵 계산2      차이
DPS             858          555          +54.4%
최종피해        780          370          +110.8%
극대율          75.0%        50.0%        +25.0%

💡 추천
🟢 계산1 추천!
극대율 풀극이 데저트이글 백상아리 크리보다 DPS가 54.4% 높습니다.
```

## 📊 계산값 비교 해석

### DPS (Damage Per Second)
- **높을수록 좋음** = 초당 더 많은 피해
- 빠른 몬스터 사냥에 유리

### 최종 피해
- **높을수록 좋음** = 한 번의 공격으로 더 많은 피해
- 보스 전투에서 중요

### 극대율
- **높을수록 좋음** = 극대 공격을 자주 함
- 극대 피해 배율과 함께 고려

### 추천 기준

| 상황 | 추천 지표 |
|------|---------|
| 몬스터 사냥 | **높은 DPS** 우선 |
| 보스 전투 | **높은 최종 피해** 우선 |
| 생존 중심 | **낮은 레벨 빌드** 선택 |
| 최고 화력 | **극대율 높은 빌드** 선택 |

## 💡 실용 팁

### 1. 같은 빌드 다른 레벨 비교
```
!계산기
→ 같은 빌드 선택
→ 다른 레벨로 또 다른 계산
→ 비교
```
**효과:** 레벨 올렸을 때 피해 증가량 확인

### 2. 같은 레벨 다른 빌드 비교
```
!계산기
→ 빌드1 선택 + 레벨 25
→ 또 다른 계산
→ 빌드2 선택 + 레벨 25
→ 비교
```
**효과:** 어떤 빌드가 더 강한지 확인

### 3. 보스별 최강 빌드 찾기
```
!계산기
→ 빌드 선택 + 레벨 40
→ 또 다른 계산 + 다른 빌드 + 레벨 40
→ 비교
```
**효과:** 보스 전투에 최적의 빌드 찾기

### 4. DPS vs 단일 피해 비교
```
!계산기로 모든 조합을 계산하여 비교
```
**효과:** 자신의 플레이 스타일에 맞는 빌드 선택

## ⚙️ 계산 원리

### 기본 대미지
```
기본 대미지 = 무기 위력 + (공격력 × 0.1) × 등급보정
```
- 신화: 1.3배
- 전설: 1.15배
- 희귀: 1.05배
- 일반: 1.0배

### 극대 피해
```
평균 피해 = (1 - 극대율) × 기본 + 극대율 × (기본 × 극대배율)
```
예) 기본 100, 극대율 50%, 극대배율 2배
- 평균 = 50% × 100 + 50% × 200 = 150

### 방어력 적용
```
최종 피해 = 평균 피해 × [100 / (100 + 적방어력 × 0.5)]
```
- 방어력이 높을수록 감소
- 방어관통으로 보정 가능

### DPS 계산
```
DPS = 최종 피해 × 공격속도
```

## ❓ 자주 묻는 질문

**Q: 두 개 이상의 빌드를 비교하려면?**
A: 한 번에 2개씩만 비교 가능합니다. 3개 비교는 두 번에 나누어 진행하세요.

**Q: 커스텀 스탯으로 계산할 수 있나요?**
A: UI 계산기는 사전 설정된 빌드만 지원합니다. 커스텀 스탯은 `!빠른계산` 명령어를 사용하세요.

**Q: 적 레벨별로 어떻게 다른가요?**
A: 
- 레벨 높을수록 방어력이 높아집니다
- 같은 빌드도 낮은 레벨 적에게는 더 많은 피해를 줍니다
- 비교할 때는 동일 레벨로 맞추세요

**Q: 극대율이 100%를 넘으면?**
A: 극대율은 보통 100% 이상이 될 수 없습니다 (게임 설정).

**Q: 어떤 빌드가 가장 강한가요?**
A: 상황에 따라 다릅니다:
- **DPS 최고:** 극대율 풀극 (858)
- **안정성:** 신화검 탱커
- **균형:** 데저트이글 백상아리 크리
